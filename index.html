Asterix_Player

Aim:

Provides an all round drop and play audio player for your website. Current version uses files ending in mp3 or MP3. Its a single file, able to self create a list of audio tracks, then assign a html 5 media player to them. The visual asthetics are dependent upon the native web browser. The php page will output a html and w3.CSS responsive webpage, that looks similar on all devices mobile to desktop.

Instructions:

  1. You must have php 5.6 or greater installed on your ftp server. See your webhost for updates.
  2. Using an FTP Client such as Filezilla, copy the file asterix_player.php into the directory that contains music you wish to share http://starmariner.net/media/other/Asterix_Player_FTP.jpg
  3. Navigate using a web browser to the asterix_player.php for example; http://starmariner.net/media/audio/Asterix_Player.php or http://http://starmariner.net/media/audio/Asterix_Player.php
  4. You should now see a list of filenames and a player under it.http://starmariner.net/media/other/Responsive_player.jpg
  5. Click on play and thats it, sorted.
